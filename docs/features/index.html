Room Control

Beautiful Touch Screen Interface

A modern Qt6-based interface for room control. Works with the official Raspberry Pi touch display or any HDMI touchscreen.

  • One-Tap Join - Join scheduled meetings with a single tap
  • Calendar View - See upcoming meetings at a glance
  • Meeting Controls - Mute, camera on/off, volume, leave meeting
  • Room Status - Show availability outside the room
